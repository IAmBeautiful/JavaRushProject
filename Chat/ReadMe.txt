Chat, JavaRush Edition.

Main class - Server.java

Control: 

Hello! First you need to start the server - class java Server.java and enter the port address (for example 9090) in the console to connect other users. Once the server is launched successfully and you see the message: "Welcome to the Server side", users can connect to the server. In the framework of the test, run the client class - mypackage / client / Client.java, after the client console enter the server address (localhost), then type the port (from the example above - 9090), then type your nickname in the chat, after which the client will be ready to work and be able to Send messages.
